# MarketShop - Kotlin/Compose

**Auteur** : Koudousse Tchedre  
**Technologie** : Kotlin avec Jetpack Compose  

## Fonctionnalités
- [x] Catalogue (API FakeStore, filtre par catégorie)
- [x] Détail produit (quantité, ajout panier)
- [x] Panier (CRUD, calcul total)
- [x] Commande (formulaire validation, sauvegarde)
- [x] Historique (liste + détails)
- [x] Profil (modification, switch thème, à propos)
- [x] Authentification locale (obligatoire)
